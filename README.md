# Toolbox

A collection of small utility scripts for development, system tasks, automation, and everyday workflows.  
This README serves as a directory of all tools in the repository.

## dev

### git
- **add-gitkeep**  
  Creates `.gitkeep` files in empty leaf directories only.

- **repo-remote-tool**  
  Converts Git remotes between HTTPS and SSH (with flip mode) for multiple repositories.
